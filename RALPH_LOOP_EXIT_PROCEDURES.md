## 🔄 EXIT SEQUENCE

Ralph Loop executes this exact sequence at exit:

### Step 1: Verify Exit Criteria (Iteration 86)

```bash
# Verify all phases complete
PHASE_1_COMPLETE=$(grep "Phase 1" ralph_loop.log | grep "COMPLETE")
PHASE_2_COMPLETE=$(grep "Phase 2" ralph_loop.log | grep "COMPLETE")
# ... etc for all phases

if [ -z "$PHASE_1_COMPLETE" ]; then
  echo "ERROR: Phase 1 not marked complete - cannot exit"
  exit 1
fi

# Verify system operational
SYSTEM_STATUS=$(curl -s https://www.doremi-live.com/api/health/live | jq .status)
if [ "$SYSTEM_STATUS" != "ok" ]; then
  echo "ERROR: System not operational - cannot exit"
  exit 1
fi

echo "Exit criteria verified ✅"
```

## 🛑 HOW TO EXIT RALPH LOOP

User simply runs:

```bash
/oh-my-claudecode:cancel
```

This will:

1. Signal Ralph Loop to stop
2. Read the exit flag
3. Mark mode as inactive
4. Clean up state files
5. Exit cleanly

If that fails:

```bash
/oh-my-claudecode:cancel --force
```

This will force immediate exit and cleanup.
